How to Install Mireka

These instructions describe setting up a single, standalone instance of Mireka. You should have have installed JDK 1.6+.
  1. Download and install Resin 4.0 snapshot (alpha) open source edition as per the Resin instructions. The Resin install directory is referenced in this document as ${resin.dir}.
  2. Download and unzip the Mireka distribution. The installation directory may be for example "C:\Program Files\Mireka" on Windows or /var/lib/mireka on Ubuntu. This directory is referenced in this document as ${mirekaHome}
  3. Configure the mirekaHome variable in Resin. To do this, edit ${resin.dir}/conf/resin.xml. After the line:
    <host id="" root-directory=".">
    insert this line:
    <resin:set var="mirekaHome" value="YOUR_DIR" />
    where YOUR_DIR should be the ${mirekaHome} directory.
  4. Copy mireka.war to ${resin.dir}/webapps/
  5. Configure Mireka by editing its configuration files in the ${mirekaHome}/conf/ directory.

    The primary configuration file is mireka.xml, which is included by mireka.war during runtime. mireka.xml in turn imports most of the other files. These configuration files will be read and processed by Resin. For more information on the syntax, see the Resin documentation. The configuration files also refer Mireka classes. For more information on Mireka classes, see the javadoc.

    logback.xml configures logging. For more information on this see the documentation of Logback.

  6. Start Resin. For more information on this, see the documentation of Resin.

To verify that everyting is going well look at the log files in ${resin.dir>/log and ${mirekaHome}/log.
